# 사후관리(PART 03) 정답 — 원문 확인 필요 문항

정답·해설은 원문 문제집(정답·해설 수록본 `의료기기인허가_문제5_사후관리.pdf`)의 페이지 하단 「정답」 띠지와 해설을 기준으로 기재했습니다.
거의 모든 문항이 원문 정답표와 일치하나, **아래 1문항만 원문 자체에 표기 불일치**가 있어 확인이 필요합니다.

## SM08-005 (이물 발견 보고 · 책 CH04 Q5)
- 문제: 의료기기 취급업체가 이물 발견 보고를 「의료기기법 시행규칙」 별지 제48호의3 서식으로 서면 보고할 경우 제출처
- **원문 「정답」 띠지: ④ (식품의약품안전처 의료기기 관리과)** ← 현재 적용
- **원문 해설 설명: ① (관할 지방식품의약품안전청 의료제품안전과)**
- 즉 원문의 정답표(④)와 해설(①)이 서로 다릅니다. 채점 표기상으로도 ①·④ 양쪽에 체크가 있어, 보유하신 원본 교재로 최종 확인 권장.
